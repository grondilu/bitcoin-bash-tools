This is a set of bash tools to manipulate bitcoin addresses, but mostly to generate keys for cold storage.

## SYNOPSIS

    $ git clone https://github.com/grondilu/bitcoin-bash-tools.git
    $ . ./bitcoin-bash-tools/bitcoin.sh
    $ newBitcoinKey
